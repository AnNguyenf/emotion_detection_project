# Emotion Detection Application
Dự án này sử dụng thư viện Watson NLP để phân tích cảm xúc từ văn bản và triển khai trên nền tảng Flask.

## Các tính năng chính:
* Phân tích cảm xúc (Joy, Anger, Fear, Sadness, Disgust).
* Giao diện web trực quan.
* Xử lý lỗi cho đầu vào trống.
